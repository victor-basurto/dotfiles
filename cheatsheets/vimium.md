# 🔥 Vimium Power User Cheat Sheet

## 🔗 Open Links

| Key  | Action               |
| ---- | -------------------- |
| `f`  | Open link            |
| `F`  | Open link in new tab |
| `yf` | Copy link URL        |

## 📜 Scrolling

| Key  | Action         |
| ---- | -------------- |
| `j`  | Scroll down    |
| `k`  | Scroll up      |
| `d`  | Half page down |
| `u`  | Half page up   |
| `gg` | Top of page    |
| `G`  | Bottom of page |

## 🔎 Search on Page

| Key | Action         |
| --- | -------------- |
| `/` | Find text      |
| `n` | Next match     |
| `N` | Previous match |

## 🌐 Navigation

| Key  | Action             |
| ---- | ------------------ |
| `H`  | Go back            |
| `L`  | Go forward         |
| `yy` | Copy page URL      |
| `p`  | Open clipboard URL |

## 📑 Tab Management

| Key  | Action             |
| ---- | ------------------ |
| `t`  | New tab            |
| `x`  | Close tab          |
| `X`  | Restore closed tab |
| `yt` | Duplicate tab      |
| `gt` | Next tab           |
| `gT` | Previous tab       |
| `^`  | Last visited tab   |

## 🔍 Search / URL Bar (Vomnibar)

| Key | Action                 |
| --- | ---------------------- |
| `o` | Search or Open URL     |
| `O` | Search/Open in new tab |
| `b` | Search bookmarks       |
| `T` | Search open tabs       |

## ✏️ Input / Forms

| Key  | Action              |
| ---- | ------------------- |
| `i`  | Enter typing mode   |
| `gi` | Jump to input field |

## 🧠 Hidden Power Commands

| Key | Action        |
| --- | ------------- |
| `[` | Previous page |
| `]` | Next page     |
| `?` | Show help     |
